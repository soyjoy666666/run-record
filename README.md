# Run Record

ランニングのベストタイムを距離ごとに管理するためのシンプルなWebアプリケーションです。

## 概要

このアプリケーションは、ランナーが異なる距離（5km、10km、ハーフマラソン、フルマラソン）ごとのベストタイムを記録・管理できるツールです。シンプルで直感的なUIで、各距離のタイムを時・分・秒で入力・編集できます。

### 主な機能

- **ホームページ**: アプリケーションの紹介と一覧ページへのナビゲーション
- **ベストタイム一覧ページ**: 距離ごとのベストタイムを入力・編集
  - 5km
  - 10km
  - ハーフマラソン
  - フルマラソン

## 技術スタック

- **フレームワーク**: Next.js 16.1.3
- **言語**: TypeScript
- **UI**: React 19.2.3
- **スタイリング**: Tailwind CSS 4
- **フォント**: Geist (Vercel)

## セットアップ

### 必要な環境

- Node.js (推奨バージョン: 20以上)
- npm

### インストール

```bash
npm install
```

## 起動方法

### 開発サーバーの起動

```bash
npm run dev
```

開発サーバーが起動したら、ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

### 本番ビルド

```bash
npm run build
```

### 本番サーバーの起動

```bash
npm run start
```

### リントの実行

```bash
npm run lint
```

## 使い方

1. アプリケーションを起動すると、ホームページが表示されます
2. 「一覧ページへ」ボタンをクリックして、ベストタイム一覧ページに移動します
3. 各距離のタイムを時・分・秒で入力・編集できます
4. 入力値は自動的にバリデーションされ、分と秒は0-59の範囲に制限されます
